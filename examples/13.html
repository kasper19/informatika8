Пример 13.4.

Написать программу для вывода изображения:

Изображение состоит из двух одинаковых фигур. Оформим вспомогательный алгоритм Figura для изображения одной фигуры. Программа:


   uses Drawman;
procedure Figura;
begin
PenDown; OnVector(1, 0);
OnVector(0, 3); 
OnVector(-1, 0);
OnVector(0, -1); 
OnVector(3, 0);
OnVector(0, 1); 
OnVector(-1, 0);
OnVector(0, -2); 
OnVector(-2, 0);
OnVector(0, -1); 
PenUp;
end;
begin
Field(10, 5);
ToPoint(1, 1); Figura;
ToPoint(6, 1); Figura;
end.

Пример 13.6.

Задана длина стороны квадрата a. Написать программу нахождения площади квадрата и длины его диагонали.

Этапы выполнения задания
I. Исходные данные: длина стороны, переменная а.
II. Результат: переменные S (площадь) и d (длина диагонали).
III. Алгоритм решения задачи.
1. Ввод исходных данных.
2. Вычисление площади по формуле S = a2.
3. Вычисление длины диагонали по формуле d = a√2.
4. Вывод результата.
IV. Описание переменных: все переменные, определенные для решения задачи, имеют тип real.
V. Программа:


var a, S, d: real;
begin
write('Введите a ='); 
read(a);
S:= sqr(a); 
d:= a*sqrt(2);
writeln('S =',s);
writeln('d =',d);
end.

Для пошуку памылак можна выкарыстоўваць сродкі наладкі праграм (гл. Дадатак).

У 1966 г. італьянскія матэматыкі Карада Бём (1923—2017) і Джузэпэ Джакапіні (1936—2001) сфармулявалі і даказалі становішча структурнага праграміравання, згодна з якім любы выконваемы алгарытм можа быць пераўтвораны да структураванага выгляду, калі ход выканання алгарытму вызначаецца пры дапамозе трох структур кіравання: паслядоўнасцей, галінаванняў і цыклаў.
